# RaiseTech公式サイト模写課題

## 概要
スクールの公式サイトを題材に、「模写」をテーマとして制作した学習課題です。  
実在のWebサイトを見本に、構造やスタイルを正確に再現する練習を目的としています。

---

## 制作期間
約1年間（学習期間含む）

---

## 使用スキル
- HTML  
- CSS / Sass  
- JavaScript  
- jQuery  

---

## 使用ツール
- Visual Studio Code  

---

## コメント
この課題では、URLだけ渡され、**デベロッパーツールの使い方も知らない状態** からスタートしました。  
「CSSの詳細度とは？」「イベントリスナーとは？」といった基礎を、調べ・試し・質問しながら理解を深めていきました。

完成までの過程で、「わからないものにどう向き合うか」という学習姿勢を身につけました。  
少しずつでも前に進めるという実感を得て、自信を持って次のステップ（WordPressやFLOCSS設計）に挑戦できるようになった作品です。
